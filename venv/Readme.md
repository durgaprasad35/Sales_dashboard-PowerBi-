📊 Sales Performance Dashboard | Power BI
📌 Project Overview

The Sales Performance Dashboard is an interactive business intelligence solution developed using Power BI to analyze sales performance across different countries, regions, states, customers, and shipping modes. The dashboard provides key business insights through dynamic visualizations, KPI cards, filters, and trend analysis, enabling data-driven decision-making.

This project demonstrates data modeling, data visualization, dashboard design, and business analytics skills using Power BI.
